what is HTML?

HTML stands for HyperText Markup Language.It is primarily used to structure and organize content of a web page.It acts as a skeleton or blueprint of every website on the internet,telling web browsershow to display text,images and links.

what is CSS?

CSS means Cascading Style Sheets.It is a style sheet language used to control the presentation, visual design and layout of web pages .CSS acts as the paint ,clothing and makeup that makes itvisually appealing.

what is javascript?

Javascript is a High-level,lightweight programming Language used to make web page interactive and dynamic

what is ReactJs?

Reactjs is a free,open-source javascript library used for building interactive user interfaces, primarily for single-page web applications.Developed by meta, its main purpose is to help developers creat fast,scalable and dynamic front-end layers.

BACK-END

what is COREJava ?

CoreJava refers to the foundational aspects of the Java programming Language, specifically matching the Java platform,standard edition.It provides the essential libraries,syntax and APIs needed to develop general-purpose,standalone and desktop-based computer applications.

what is Advance Java?

Advance Java refers to the specialized area of the Java programming language used to build web-based,distributed and large-scale enterprise applications.

DATABASE MANAGEMENT

what is MYSQL ?

MYSQL is a free,open-source relational database management system used to store,organize and manage data.It is developed and maintained by the Oracle Corporation.

what is SpringBoot ?

Spring Boot is an open-source,java-based framework uses to build standalone, applications with minimal manual configuration.It is an extension built on top of the broader spring Framework,designed to eliminate tedious boilerplate code and complex XML setups.
